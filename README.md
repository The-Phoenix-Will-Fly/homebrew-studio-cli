# Homebrew Tap for Tokens Studio CLI

This is the official Homebrew tap for [Tokens Studio CLI](https://github.com/The-Phoenix-Will-Fly/studio-on-rails/tree/main/packages/cli).

## Installation

```bash
brew install The-Phoenix-Will-Fly/studio-cli/studio
```

Or add the tap first:

```bash
brew tap The-Phoenix-Will-Fly/studio-cli
brew install studio
```

## Updating

```bash
brew upgrade studio
```

## About

The Tokens Studio CLI is a fast, secure command-line tool for syncing design tokens from Tokens Studio to your codebase.

- **Zero Dependencies**: Single binary, no Node.js required
- **Cross-Platform**: macOS (Intel & Apple Silicon), Linux
- **Secure**: SOC2 compliant with OAuth 2.0 authentication
